# Winter — voice clips

Drop raw voice clips for this character in this folder. **Any format works** —
`.mov`, `.mp4`, `.mp3`, `.m4a`, `.wav`, `.flac`.

These are **source material**. They get processed into `../reference.wav`,
the single clean clip Winter clones the voice from.

Tips:
- Clean speech works best — no background music or sound effects.
- One good ~10–20 second clip is enough; Winter does zero-shot cloning.
